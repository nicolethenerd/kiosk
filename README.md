# Family Kiosk

A touch-friendly family dashboard built with Next.js. Provides quick access to shared household apps: recipes, calendar, shopping list, chores, medications, and contacts.

## Prerequisites

- [Node.js](https://nodejs.org/) v18 or later
- npm (included with Node.js)

## Installation

```bash
npm install
```

## Running the Dev Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Running Tests

```bash
npm test
```

## Building for Production

```bash
npm run build
npm start
```
